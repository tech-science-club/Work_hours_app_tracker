# Work Hours App

Qt Quick and C++ interaction may help to ideas come true 

Main aim of this App is to collect information regarding user's hours at work place.
Simple interface allows to input start working days combining it with following breakes.
Information is being saved in local gadjet's direcrories (check mainwindow::initializePath())
2 popup dialog windows can help user to observe timing information during current day and
during whole month depicting bar chart.
Information is being saved for each day and there is a posibilty to save txt file on the disc
with information in [day, hours] view.
